====== 0. What are classes ======
Created Thursday 07 January 2021

Java classes are some of the core building blocks of Java applications, toolkits, frameworks, APIs etc.

A Java ''class'' can contain the following building blocks:
* Fields
* Constructors
* Methods
* Nested Classes
